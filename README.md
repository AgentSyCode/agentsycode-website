# AgentSyCode - موقع شركة تقنية متكامل

موقع حديث ومتجاوب لشركة AgentSyCode، مع بنية مشروع متكاملة تستخدم GitHub، Vercel، MongoDB، وSupabase.

## المميزات

- ✨ تصميم حديث ومتجاوب مع جميع الأجهزة
- 🎨 ألوان رئيسية: #fccf00, #6dc1e5
- 📱 شريط تنقل سلس مع قائمة هامبرغر
- 🚀 قسم هيرو مع خلفية وأزرار تفاعلية
- 💎 ثلاثة بطاقات مميزات مع أيقونات دائرية متدرجة
- 📲 قسم تحميل التطبيق
- 📞 قسم تواصل مع إمكانية نسخ البريد
- 🔧 بنية مشروع منظمة

## التقنيات المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Fonts**: Google Fonts (Cairo)
- **Icons**: Font Awesome 6
- **Hosting**: Vercel
- **Version Control**: GitHub
- **Database**: MongoDB + Supabase
